In order to become a better dev in the sense of tackling bigger projects and larger codebases. It is important to leverage tools to become more productive.
During university, we were taught to leverage unix commands to do useful functions for whatever task we need it. In modern day, while those "legacy" tools 
are indeed useful and preset on many systems, there are other popular tools created by the community that offer better experiences with better performance and the same functionality.

I will provide a list of what I feel are the essential programs and also notable ones which I feel are probably "nice to haves" outside of the core tools.

## Essentials
- grep -> rg
- find -> fd
- sed -> sd
- cat -> bat
- htop -> btm/btop
- ls -> lsd
- du -> dust/gdu
- man -> tealdear
- vi -> neovim
- tree

## Nice to Haves
- tree
- delta (git diff)
